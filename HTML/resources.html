Ableton Live is a digital audio workstation designed to be an instrument for live performances as well as a tool for composing, recording, arranging, mixing, and mastering. It is also used by DJs.

Pro Tools is another digital audio workstation developed and released by Avid Technology which can be used for a wide range of sound recording and sound production purposes.

Max is a visual programming language for music. It is used by composers, performers, software designers, researchers, and artists to create recordings, performances, and installations.

Csound

Csound is a Domain-specific computer programming language for audio programming. It is called Csound because it is written in C, as opposed to some of its predecessors.

A synthesizer is an electronic musical instrument that generates audio signals. Synthesizers are typically sequenced, controlled by software or instruments via MIDI, or played with keyboards or other controllers.

ChucK is a concurrent, strongly timed audio programming language for real-time synthesis, composition, and performance. It supports concurrency and multiple, simultaneous, dynamic control rates.
